# Global life-history reference table

A reference dataset of life-history traits for marine taxa used by
CheckEM.

## Usage

``` r
global_life_history
```

## Format

A data frame with one row per taxon (typically species) and columns for
taxonomic identity and one or more life-history attributes.

## Source

Compiled by the CheckEM team from global life-history reference sources.

## Details

This dataset may be used when Australian-specific values are
unavailable, or for global workflows and comparisons.
