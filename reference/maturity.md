# Maturity reference table

A reference dataset of maturity-related information (e.g.
length-at-maturity or maturity status categories) used by CheckEM.

## Usage

``` r
maturity
```

## Format

A data frame with one row per taxon (or taxon-region combination) and
columns describing maturity metrics or categories.

## Source

Compiled by the CheckEM team from maturity reference sources.

## Details

Used for QA/QC checks and/or derived summaries where maturity
information is relevant.
