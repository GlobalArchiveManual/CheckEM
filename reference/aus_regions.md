# Australian marine regions

A reference table of Australian marine regions used by CheckEM for
lookups, grouping, and reporting.

## Usage

``` r
aus_regions
```

## Format

A data frame with one row per region and one or more columns describing
the region name/identifier and related metadata.

## Source

Compiled by the CheckEM team from Australian marine region frameworks.

## Details

This dataset is used internally by CheckEM functions and/or the bundled
Shiny app to standardise region names and region identifiers.
