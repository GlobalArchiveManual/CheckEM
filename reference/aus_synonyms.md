# Australian species synonym table

A lookup table of taxonomic synonyms used by CheckEM to standardise
species names.

## Usage

``` r
aus_synonyms
```

## Format

A data frame with one row per synonym relationship and columns
containing at least a synonym and an accepted (preferred) name.

## Source

Compiled by the CheckEM team from taxonomic reference sources.

## Details

This dataset helps harmonise species naming across datasets and
annotations.
