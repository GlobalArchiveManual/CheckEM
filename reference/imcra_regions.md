# IMCRA regions reference table

A reference table describing IMCRA (Integrated Marine and Coastal
Regionalisation of Australia) regions used by CheckEM.

## Usage

``` r
imcra_regions
```

## Format

A data frame with one row per IMCRA region and columns describing region
identifiers, names, and related metadata.

## Source

IMCRA framework as compiled for use in CheckEM.

## Details

Used for region-based grouping, validation, and reporting.
