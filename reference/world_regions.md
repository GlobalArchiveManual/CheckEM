# World marine regions reference table

A reference table of world marine regions used by CheckEM for global
grouping, mapping, or reporting.

## Usage

``` r
world_regions
```

## Format

A data frame with one row per region and one or more columns describing
the region name/identifier and related metadata.

## Source

Compiled by the CheckEM team from global marine region frameworks.

## Details

This dataset may include region names in languages other than English.
If you see R CMD check warnings about non-ASCII strings in this dataset,
ensure it is saved with UTF-8 encoding (or consider transliteration if
strict ASCII is required).
