# CATAMI classification reference table

A reference table containing CATAMI (Collaborative and Automated Tools
for Analysis of Marine Imagery) classes and/or mappings used by CheckEM.

## Usage

``` r
catami
```

## Format

A data frame describing CATAMI class names/codes and any hierarchical
relationships or descriptors included in the CheckEM distribution.

## Source

CATAMI classification framework as compiled for use in CheckEM.

## Details

Used to support habitat / benthic classification validation and
standardisation.
